When you need to change some text in the section - open the file, find needed section and change the text that you want. You also can use option "find" in your code editor to quickly find the text.

CSS+SASS guide #back to top

All CSS files are in the folder css. Here you can find main.css and main-*.css files with our template styles and vendor CSS files (bootstrap, OWL carousel, lightbox and other).

main.css its a file with default color scheme. If you want to use another color scheme, you need to connect the corresponding file (one of main-*.css) instead of main.css. Note: you need only one file with name main to be connected to the site. They are similar, but with different colors. All colors correspond to the file's name.

For example:

if you want to use green color on the site, connect the main-green.css as a style file on your site.

Structure of the sass folder

If you are familar with SASS and want to change something you can use our scss sources.

  • base - includes base styling (colors, fonts, typo etc)
  • components - includes reusable components (buttons, forms, timeline etc)
  • modules - styling for sections
  • utilites - includes mixins and breakpoints
  • main-*.scss - style files for another colors
  • main.scss - default style file

For sass files we use the mixin approach. In it the specific file its a mixin that you can import where you need.

For example:

If you need to add another module, use this template

					@mixin module-your-name {
						/* your SASS code here */
					}

And then import it in the _modules.scss file.

JS guide #back to top

This template use our JS file main.js and other vendor scripts such as bootstrap.min.js, jquery.min.js and other.

Structure of main.js file

You can find this file in the folder "js".

For all specific sections we use functions and then init them. So if you want to add some new functionality we recommend create a new function for this and init it in the end of the main.js file.

You onle need to change date in Countdown.

Countdown:

  1. Open the file main.js
  2. Find the section "Countdown"
  3. In the bottom of this section you'll find the line with the date. This is the date for wedding. 
    						// set your wedding date here
						var deadline = 'December 17 2017 17:30:00 GMT+0300';
  4. Change this date to which you want.
  5. Be careful! Change only the date in the quotes in this format 'Month day year time(optional)'.
    Example:
    var deadline = 'December 17 2016 17:30:00 GMT+0300';
    var deadline = 'December 17 2016';
